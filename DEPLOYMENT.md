# Deployment (Cloudflare Pages + GitHub)

- **Host:** Cloudflare Pages, connected to your **`Focushift-website`** GitHub repo (or whatever repo you use).
- **Build:** None — static HTML at repo **root**.
- **Output directory:** `/` (root), **not** a `website` subfolder.
- **Canonical domain:** `https://focusshift.app/`

See **`PUSH_TO_GITHUB.md`** in this folder for copy-paste commands to copy these files into that repo and push.
