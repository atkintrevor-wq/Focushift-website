# Deployment (Cloudflare Pages + GitHub)

- **Host:** Cloudflare Pages, connected to your **`Focushift-website`** GitHub repo.
- **Build:** None — static HTML at repo **root**.
- **Output directory:** `/` (root), **not** a `website` subfolder inside another project.
- **Canonical domain:** `https://focusshift.app/`

**How to publish changes:** use **[DEPLOY_CHEATSHEET.md](./DEPLOY_CHEATSHEET.md)** (copy/paste Git commands from inside this `website` folder).
